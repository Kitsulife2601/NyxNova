# Datenschutzerklärung für Novaaddon

Stand: 11. Juni 2026

Novaaddon ist eine Browser-Erweiterung, die den neuen Tab durch eine persönliche Startseite mit Uhr, Suche, Lesezeichen, Hintergrund, optionaler Webseiten-Übersetzung und optionaler Discord-Herz-Funktion ersetzt.

## Welche Daten verarbeitet werden

Novaaddon speichert Einstellungen lokal im Browser, damit die Startseite beim nächsten Öffnen wieder gleich aussieht. Dazu gehören Uhr-Design, Schriftart, Hintergrundauswahl, eigene Lesezeichen, Übersetzungsoptionen und optional eine gespeicherte Discord-ID.

Wenn die automatische Webseiten-Übersetzung aktiviert ist, verarbeitet Novaaddon die URL der geöffneten Webseite, um diese Webseite über Google Translate zu öffnen. Novaaddon speichert keinen Browserverlauf und erstellt keine Besuchslisten.

Wenn die Discord-Herz-Funktion genutzt wird, kann eine Discord-ID lokal gespeichert und an einen lokal laufenden Bot-Server auf `127.0.0.1:8787` gesendet werden. Diese Funktion ist optional und funktioniert nur, wenn der Nutzer den lokalen Bot-Server selbst startet.

## Welche Daten nicht verarbeitet werden

Novaaddon liest keine Webseiteninhalte aus. Es werden keine Texte, Bilder, Videos, Passwörter, Zahlungsdaten, Standortdaten, Gesundheitsdaten oder Finanzdaten gesammelt.

Novaaddon überwacht keine Mausbewegungen, Tastatureingaben, Scrollbewegungen oder sonstige Nutzeraktivitäten auf Webseiten.

## Weitergabe von Daten

Novaaddon verkauft keine Nutzerdaten und gibt keine Nutzerdaten für Werbung, Profilbildung oder Kreditwürdigkeitsprüfungen weiter.

Bei aktivierter Übersetzung wird die aufgerufene Webseiten-URL an Google Translate übergeben, damit die Übersetzungsseite geöffnet werden kann.

Bei aktivierter Discord-Herz-Funktion werden die eingegebene Discord-ID und die Herz-Nachricht nur an den lokal laufenden Bot-Server auf dem eigenen Gerät gesendet.

## Speicherung

Einstellungen werden lokal im Browser gespeichert. Eigene Hintergründe und eigene Lesezeichen können lokal im Browser gespeichert werden, damit sie auf der Startseite erhalten bleiben.

## Kontakt

Bei Fragen zur Erweiterung oder zum Datenschutz kann der Publisher über die im Chrome Web Store angegebene Kontakt-E-Mail erreicht werden.
